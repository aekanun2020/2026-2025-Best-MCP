# n8n Workflows

This directory contains n8n workflow JSON files for integrating with PyRAGDoc MCP Server.

## Usage

1. Import the JSON file into your n8n instance
2. Configure the HTTP Request nodes to point to your PyRAGDoc server (default: `http://localhost:8000`)
3. Update any credentials or environment variables as needed
