## 📋 Tab 1: รายการเอกสาร

### ฟีเจอร์
- แสดงรายการไฟล์ .md ทั้งหมดใน `corpus_input/`
- แสดงข้อมูล:
  - 📄 ชื่อไฟล์
  - 📊 ขนาดไฟล์ (KB/MB)
  - 🕐 วันที่แก้ไขล่าสุด
- ปุ่ม 🗑️ ลบเอกสาร (แต่ละไฟล์)

### ตัวอย่างการแสดงผล
```
📄 1.md          64.0 KB    2026-04-14 10:30:00    [🗑️]
📄 2.md          60.0 KB    2026-04-14 10:31:00    [🗑️]
📄 44.md         49.0 KB    2026-04-14 10:32:00    [🗑️]
📄 5555.md       59.0 KB    2026-04-14 10:33:00    [🗑️]
```

### การลบเอกสาร
1. คลิกปุ่ม 🗑️ ข้างไฟล์ที่ต้องการลบ
2. ไฟล์จะถูกลบทันที
3. หน้าจะ refresh อัตโนมัติ

## ➕ Tab 2: เพิ่มเอกสาร

### วิธีที่ 1: อัปโหลดไฟล์ 📤

**ขั้นตอน:**
1. เลือก "📤 อัปโหลดไฟล์"
2. คลิก "Browse files"
3. เลือกไฟล์ .md จากเครื่อง
4. ดูตัวอย่างเนื้อหา
5. คลิก "💾 บันทึก"

**รองรับ:**
- ✅ ไฟล์ .md เท่านั้น
- ✅ Encoding: UTF-8
- ✅ ขนาดไฟล์: ไม่จำกัด

**ตัวอย่าง:**
```
[Browse files] → เลือก my-document.md
↓
[Preview]
# หัวข้อเอกสาร
เนื้อหา...
↓
[💾 บันทึก] → ✅ บันทึกเอกสารสำเร็จ: my-document.md
```

```yaml
  - ./AuthenticRAG-Qwen2.5API:/app
  - ./AuthenticRAG-Qwen2.5API/corpus_input:/app/corpus_input  # Read-write
```

## 🐛 Troubleshooting

### ปัญหา: ไม่เห็นไฟล์ที่เพิ่ม
**สาเหตุ:** Volume mount ไม่ถูกต้อง

**แก้ไข:**
```bash
# ตรวจสอบ volume
docker exec contextual-rag-ui ls -la /app/corpus_input/

# ถ้าไม่เห็นไฟล์ ให้ restart container
docker-compose restart streamlit
```

### ปัญหา: Permission denied
**สาเหตุ:** ไม่มีสิทธิ์เขียนไฟล์

**แก้ไข:**
```bash
# ตรวจสอบ permission
ls -la AuthenticRAG-Qwen2.5API/corpus_input/

# แก้ไข permission
chmod 755 AuthenticRAG-Qwen2.5API/corpus_input/
```

### ปัญหา: ไฟล์ไม่ถูก index
**สาเหตุ:** ยังไม่ได้ run indexing

**แก้ไข:**
```bash
# Index ผ่าน MCP tool
curl -X POST http://localhost:9001/mcp \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "tools/call",
    "params": {
      "name": "add_documents",
      "arguments": {
        "path": "/app/rag/corpus_input"
      }
    }
  }'
```
