# สรุปการประเมินคำตอบจากระบบ AuthenticRAG

## คำแนะนำการประเมิน
- **(1) ตอบถูกแต่ไม่ครบ** = ข้อมูลถูกต้องแต่ขาดรายละเอียดสำคัญ
- **(2) ตอบถูกและครบถ้วนสมบูรณ์** = ข้อมูลถูกต้อง ครบถ้วน ตรงประเด็น
- **(3) ตอบไม่ถูก** = ข้อมูลผิด หรือไม่ตรงคำถาม
